## 🌟 **Option 2: Vercel (Serverless)**

### Why Vercel?
- ✅ Global edge network
- ✅ Automatic scaling  
- ✅ Good for lighter APIs

### Deploy to Vercel:
1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel --prod`
3. Or use GitHub integration

**Your API URL:** `https://your-project-name.vercel.app/hackrx/run`
